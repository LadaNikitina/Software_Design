# Карта

## Обозначение клеток
* Чёрные клетки - стены
* Зелёные клетки - колючие лозы
* Красные клетки - лава

## Параметры генерации
* Размер карты
* Вероятность клетки с лавой
* Вероятность клетки с колючей лозой
